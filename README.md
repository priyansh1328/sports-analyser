# Cricket & Football Stats Analyser

Analyses IPL and Premier League statistics
with charts, tables and CSV exports.

## How to Run
pip3 install -r requirements.txt
python3 main.py

## What it does
- Shows IPL batting and bowling stats
- Shows Premier League scoring stats
- Saves 8 charts to charts/ folder
- Saves data to CSV files in data/ folder
- Saves logs to logs/ folder
